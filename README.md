# Верстка первой страницы "ЭпплShop"

### Установка
Сначала необходимо установить все библиотеки:

1.
```sh
$ npm i
```
2.
```sh
$ bower i
```
Перед началом пользования запустите скрипт для компиляции кода:
```sh
$ gulp build
```
Чтоб запустить локальный сервер: 
```sh
$ gulp connect
```
